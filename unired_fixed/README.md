# UniRed — Университетская платформа

Reddit-подобная платформа для студентов. Учебный проект.

## Стек
- Python (Flask) — бэкенд
- SQLite — база данных (файл database.db)
- HTML + CSS + JavaScript — фронтенд
- Jinja2 — шаблоны

## Установка и запуск

```bash
# 1. Установить зависимости
pip install flask

# 2. Запустить
python app.py

# 3. Открыть в браузере
http://localhost:5000
```

## Функционал
- Регистрация и вход
- Лента постов (горячее / новое / топ)
- Категории: новости, обсуждение, вопрос, ивент, объявление
- Голосование (upvote)
- Комментарии к постам
- Страница ивентов и событий
- Поиск по постам
- Фильтрация по категориям
- Закреплённые посты (для admin/moderator)

## Структура проекта
```
unired/
├── app.py              # Flask приложение, все маршруты
├── database.db         # SQLite база (создаётся автоматически)
├── requirements.txt    # Зависимости
├── templates/
│   ├── base.html       # Базовый шаблон (навигация, стили)
│   ├── index.html      # Главная страница / лента
│   ├── post.html       # Страница поста с комментариями
│   ├── new_post.html   # Форма создания поста
│   ├── events.html     # Страница ивентов
│   ├── new_event.html  # Форма добавления ивента
│   ├── login.html      # Вход
│   └── register.html   # Регистрация
└── README.md
```
